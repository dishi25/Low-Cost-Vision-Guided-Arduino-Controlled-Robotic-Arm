## <img src="https://cdn-icons-png.flaticon.com/512/3135/3135715.png" width="22"> Software Simulation & Hardware Prototype

This project includes both a **software-level simulation environment** and a **physical robotic arm prototype** developed for demonstrating real-time vision-guided object sorting.

The simulation was designed to test the object detection pipeline, serial communication workflow, and robotic control logic before deploying the system onto hardware.

---

## <img src="https://cdn-icons-png.flaticon.com/512/2921/2921222.png" width="22"> Software Simulation

The software simulation demonstrates:

- Real-time webcam streaming
- Color-based object detection using OpenCV
- Object coordinate tracking
- Detection of red and green objects
- Vision-guided object selection
- Interactive control interface
- Python-Arduino communication workflow simulation

The simulation environment was used to validate the detection accuracy and robotic response logic prior to hardware implementation.

---

## <img src="https://cdn-icons-png.flaticon.com/512/4712/4712109.png" width="22"> Hardware Prototype

The repository also contains images and media of the physical robotic arm prototype developed during the project.

The hardware prototype includes:

- Arduino-controlled robotic arm
- Servo motor actuation
- Embedded control system
- Vision-guided object sorting mechanism
- Integrated webcam-based detection setup

The physical implementation demonstrates the practical integration of:

- Computer Vision
- Embedded Systems
- Robotics & Automation
- Real-Time Control Systems

## <img src="https://cdn-icons-png.flaticon.com/512/3523/3523887.png" width="22"> Purpose of the Simulation

The simulation phase was essential for:

- Testing computer vision algorithms
- Verifying object classification logic
- Validating communication between Python and Arduino
- Reducing hardware debugging complexity
- Improving robotic movement accuracy
- Optimizing system workflow before deployment

This workflow reflects real-world engineering practices used in robotics and industrial automation system development.
